# AI Chatbot Foundations

A simple OpenAI-powered chatbot project built to learn LLM fundamentals, prompt structure, conversational memory, token usage, and clean Python project organization.

## Features

- OpenAI chat completions
- System prompt support
- Multi-turn conversation memory
- Token usage tracking
- Chat log saving
- Clean project structure

## Tech Stack

- Python
- OpenAI API
- python-dotenv
- Jupyter Notebook
